## MySQL 通配符与正则表达式

***1、LIKE操作符***

为在搜索子句中使用通配符，必须使用like操作符。

> 注意：操作符何时不是操作符？答案是它在作为谓词时。

***2、百分号（%）通配符***

% 表示任何字符出现任意次数。

```mysql
MariaDB [test]> select id,age,province
-> from user
-> where province like '天%'
-> ;
+----+------+----------+
| id | age | province |
+----+------+----------+
| 3 | 56 | 天津 |
| 9 | 33 | 天津 |
+----+------+----------+
2 rows in set (0.00 sec)
```
***3、下划线（_）通配符***

下划线_，只匹配单个字符而不是多个字符。

使用通配符的技巧：
1.不要过度使用通配符
2.在确实需要使用通配符时，不要用在搜索模式的开始处。影响效率
3.仔细注意通配符的位置
