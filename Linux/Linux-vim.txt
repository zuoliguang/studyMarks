linux 对 vim、vi 编辑器的操作

注：vim 作为 vi 的升级版本，已经兼容vi所有的操作，这里只讲vim


0、行号
	: set nu 	显示行号
	: set nonu 	取消显示行号


1、光标移动
	$ 	将光标移动到 当前行 的末尾
	H 	将光标移动到 当前页 的第一行起始位置
	L 	将光标移动到 当前页 的最后一行的起始位置
	gg 	将光标移动到 文档的 开始处
	G 	将光标移动到 文档的 末尾
	<数字>G 将光标移动到 输入的行数


2、保存 与 退出
	:q 	退出
	:q! 舍弃修改 强制退出
	:w 	保存
	:wq 保存退出


3、删除、恢复
	dd 			删除光标所在行
	<数字>dd 	删除从光标当前行开始指定的数字行
	u 			恢复刚被删除的文本
	U 			恢复光标所在行在此次的编辑中的全部改变


4、复制、粘贴
	yy 			复制光标所在行 
	<数字>yy 	复制光标所在行及往下[数字]行 
 	p 			粘贴复制的内容到光标所在的下一行 


5、查找字符
	/<字符串> 	从光标所在行往 后 查找符合的字符串
	?<字符串> 	从光标所在行往 前 查找符合的字符串
	n 			下一个
	N 			上一个
